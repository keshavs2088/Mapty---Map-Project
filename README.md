Mapty – Workout Tracker

Mapty is a map-based workout tracking application built with JavaScript and Leaflet.js.

The application allows users to:
-Track running and cycling workouts
-Place workout markers on an interactive map
-Store workout information dynamically
-Move the map to saved workout locations
-Delete workouts and markers
-Use custom workout markers and popup styling

Features
📍 Interactive map using Leaflet.js
🏃 Running workout tracking
🚴 Cycling workout tracking
📌 Custom workout markers
🗺️ Smooth map navigation to workout locations
📝 Dynamic workout records
❌ Delete workouts and markers
🎨 Custom popup styling
📱 Responsive UI layout
